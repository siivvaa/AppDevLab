Make a webpage that counts the number of times it has been visited using a storage.

Tech used - MongoDB, NodeJS.

Instructions

!! Make sure you have NodeJS installed and a MongoDB cluster up and running.

1) Download all files.
2) npm i
3) In line 21 of app.js, replace <username>, <password> and <clustername> with the respective details of the cluster that you have created.
4) npx nodemon app.js
5) In Chrome, go to 127.0.0.1:8000/
