1) Unzip all files and "node i" to install all packages.
2) Make a folder called "fetchedData" in this directory. This is the storage location of all the fetched files from the MongoDB databse.
3) Make sure the dbUri is correct or not. (depending on whether it is a local instance or not).
4) "npx nodemon app.js" to start the server.
5) Open "http://127.0.0.1:3000/".
